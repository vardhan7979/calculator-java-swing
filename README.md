# Java Swing Calculator

## Overview
This project is a simple Java Swing GUI Calculator that performs basic arithmetic operations.  
It demonstrates the use of Swing components and event handling in Java.

The application allows a user to enter two numbers and perform:
- Addition
- Subtraction
- Multiplication
- Division

The result is displayed dynamically in the GUI.

## Technologies Used
- Java
- Java Swing (GUI)
- NetBeans IDE

## Features
- Graphical User Interface using Swing
- Input using JTextField
- Buttons for arithmetic operations
- Event handling using ActionListener
- Result displayed instantly in the result field

## GUI Components Used
- JFrame
- JLabel
- JTextField
- JButton
